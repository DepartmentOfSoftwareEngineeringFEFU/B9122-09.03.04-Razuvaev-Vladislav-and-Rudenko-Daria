# B9122-09.03.04-Razuvaev-Vladislav-and-Rudenko-Daria
Программная система выявления рисков в составе программного обеспечения с помощью инструментов SBOM

О чём приложение

SBOM Analyzer — десктопное приложение для проверки безопасности программных зависимостей. Вы загружаете SBOM-файл (список всех библиотек вашего проекта), а приложение находит уязвимости в каждой из них, показывает граф зависимостей и объясняет что именно нужно исправить и в какой срок.

SBOM (Software Bill of Materials) — это структурированный список всех компонентов программного продукта: библиотек, фреймворков и их версий. Такой файл можно сгенерировать инструментами Syft, Grype или встроенными средствами сборки.

Что делает приложение?

Читает SBOM-файл в форматах CycloneDX (JSON или XML) и SPDX (JSON или XML)
Проверяет каждую библиотеку через открытую базу уязвимостей OSV.dev, которая агрегирует данные из NVD, GitHub Advisories и других источников
Строит граф зависимостей — показывает как библиотеки связаны между собой и где именно находится уязвимость
Формирует рекомендации — для каждой уязвимости указывает конкретное действие (обновить, применить workaround, заменить), оценку трудозатрат и срок устранения (SLA)
Экспортирует отчёт в форматах TXT, JSON или HTML

команды:
